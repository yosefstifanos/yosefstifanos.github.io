Model comparison

Log loss on held-out seasons 2018–2025 via forward-chaining cross-validation. Lower = better.

Why ensemble beats single models

Blending XGBoost, a neural net, random forest, and Bayesian logistic regression reduces out-of-sample log loss by ~0.02 vs. the best single model. Each architecture overfits to different spurious patterns — averaging them out cancels the noise. Weights assigned by minimizing held-out log loss across 8 validation seasons.

Training pipeline

End-to-end: data ingestion → feature engineering → cross-season validation → ensemble → calibration → simulation.

Validation strategy

Forward-chaining cross-validation: Train on seasons 2000–N, validate on season N+1. Repeat for N = 2017 to 2024. Prevents any lookahead bias. Final model trained on full 2000–2025 data.

Probability calibration: Isotonic regression on held-out seasons. When the model predicts 70%, the team wins ~70% historically. Evaluated with Brier score and reliability diagrams.

Monte Carlo simulation: 100,000 independent full-tournament paths. Each game sampled from calibrated win probabilities. Aggregates round-reach rates, Final Four frequencies, and championship odds.
